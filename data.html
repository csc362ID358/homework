Welcome! This webpage defines and gives examples of the four data types: Nominal, Interval, Ordinal, and Ratio data.

Nominal

Nominal data types are data where the data described has no real numerical value and cannot be ordered. Nominal data could be a list of the favorite fruit of every person in a class, or the hair colors of members of a family.

Ordinal

Ordinal data types are data that can be ordered, but the difference between data points is not known. The answers that patients give when their doctor asks them to rate their pain on a scale of 1-10 would be a common example of an ordinal data set.

Interval

Interval data types are data that can be ordered, and the differences between data points are known, but there is no true zero. You can add or subtract one value from another value, but you can't multiply values or understand values as a ratio between one value and another. An example of interval data would be a dataset containing the high and low temperatures for a month, measured on a Fahrenheit or Celsius scale.

Ratio

Ratio data types are data that can be ordered, that has differences that are known, and that has a true zero. You can multiply and divide values and the relationships between values stay the same. A dataset of the weight and height of every member of an age group would be a ratio data.
